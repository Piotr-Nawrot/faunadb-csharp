\doxysubsection{Detailed Description}
Represents any scalar or non-\/scalar value in the \mbox{\hyperlink{namespace_fauna_d_b}{Fauna\+DB}} query language. \mbox{\hyperlink{namespace_fauna_d_b}{Fauna\+DB}} value types consist of all of the J\+S\+ON value types, as well as the Fauna\+D\+B-\/specific types, \mbox{\hyperlink{class_fauna_d_b_1_1_types_1_1_ref_v}{RefV}} and \mbox{\hyperlink{class_fauna_d_b_1_1_types_1_1_set_ref_v}{Set\+RefV}}. 

Scalar values are \mbox{\hyperlink{class_fauna_d_b_1_1_types_1_1_long_v}{LongV}}, \mbox{\hyperlink{class_fauna_d_b_1_1_types_1_1_string_v}{StringV}}, \mbox{\hyperlink{class_fauna_d_b_1_1_types_1_1_double_v}{DoubleV}}, \mbox{\hyperlink{class_fauna_d_b_1_1_types_1_1_boolean_v}{BooleanV}}, \mbox{\hyperlink{class_fauna_d_b_1_1_types_1_1_null_v}{NullV}}, \mbox{\hyperlink{class_fauna_d_b_1_1_types_1_1_ref_v}{RefV}}, and \mbox{\hyperlink{class_fauna_d_b_1_1_types_1_1_set_ref_v}{Set\+RefV}}. 

Non-\/scalar values are \mbox{\hyperlink{class_fauna_d_b_1_1_types_1_1_object_v}{ObjectV}} and \mbox{\hyperlink{class_fauna_d_b_1_1_types_1_1_array_v}{ArrayV}}.
